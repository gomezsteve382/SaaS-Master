content hash against the existing top-level `attached_assets/*.bin` / `*.BIN`

filenames are preserved; there were no filename collisions with the existing
corpus. Four of the 43 new dumps are byte-identical to a sibling inside the
same bundle (distinct filenames, e.g. `*_DONTMATCH2`); both copies are kept
because each is "new" relative to the existing corpus.

immobilizer EEPROMs, Continental GPEC2A PCM EEPROMs, and a 2022 Charger Redeye
6.2 / 797 BCM+RFHUB pair) now visible to every analyzer, the fixture loader,
and the RFHUB index-crack harness via the asset sweep.

Only the `eeprom-dumps/` binaries were staged. The bundle's `databases/`,
`algorithms-*`, and `analysis-docs/` parts are out of scope for this note.
