UX Audit - May 17, 2026

HOME PAGE:
- Upload zone is clear
- Stats show 0 for everything (no analyses yet)
- Nav has: Multi-Align, Compare & Patch, Vault, Patterns, KG
- Intelligence Search and Vault sections visible

PROBLEMS IDENTIFIED:
1. After uploading a binary, user lands on Analysis page with TABS: Overview, Swarm Intel, Chat, Tool Call Trace
2. Chat is BURIED as the 3rd tab - user has to know to click it
3. The Analysis page shows raw technical data (algorithms, seed keys, etc.) without explaining WHAT it means
4. No clear "talk to the AI about this file" CTA
5. The user's preference is: upload → state goal → AI does the work
6. User wants a CONVERSATIONAL interface, not a data dump

FIX PLAN:
- Make Chat the PRIMARY view on Analysis page (not a tab)
- Show a split layout: Chat on left/bottom, findings summary on right/top
- Add a persistent "Ask VENOM" floating button
- Reduce tab clutter - merge Swarm Intel into Overview
- Add clear section headers explaining what each finding means
